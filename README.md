# parthenon

Parthenon AMR infrastructure 

# Required Dependencies

* CMake 3.13 or greater
* gcc or intel compiler
* mpi 
* openMP
* hdf5

# Dependencies Other

* catch2 

# Installation

    mkdir build
    cd build
    cmake ../
    make
    make test
